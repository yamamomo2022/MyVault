
# Title
- 緑の豊かさエージェント
- NDVIAgent

# concept
- 住んでいるところや旅行先の緑の豊かさを、衛星データを元に算出して教えてくれるAIAgentを廃初する。衛星データの活用を体験できるアプリ。

	- Flutter
- バックエンド

	- google earth engine api
	- Agent Development Kit
